# Team Score Calculator for Hult Prize

## Overview

This is a **Team Score Calculator** built using Python and Tkinter. It allows users to input scores for different teams and calculate their total score based on four criteria. The application sorts teams based on their scores and allows exporting the results to a CSV file. It is specifically designed to streamline scoring for competitions like the **Hult Prize at Galala University**.

## Features

- Input team names and scores for four criteria.
- Validate that scores are between 0 and 20.
- Display sorted team rankings.
- Save the sorted results as a CSV file.
- Real-time updates on the number of teams entered.
- Simple and user-friendly interface.

## How to Use

1. Enter the team's name and their scores for the four criteria.
2. Click `Enter` to add the team's total score.
3. Click `Sort` to display the teams ranked by total score.
4. Use `Save In CSV` to export the results to a CSV file named `Teams.csv`.

## Installation

1. Clone the repository or download the code.
2. Ensure you have Python installed along with the following dependencies:
   - Tkinter (pre-installed with Python)
   - pandas
   - numpy

3. Run the `app.py` file to start the application.
